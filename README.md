# 🌌 Junaid Muhammad Khan | Professional Portfolio

A high-performance, responsive portfolio designed with a "Deep Space" aesthetic. This site serves as the central hub for my game development deployments, software engineering projects, and professional contact point for App Store support.

**Live Demo:** [https://junaid-muhammad-khan.github.io/](https://junaid-muhammad-khan.github.io/)

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Glassmorphism & Animations)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) (via CDN)
* **Animations:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
* **Interactive Elements:** Vanilla JavaScript & HTML5 Canvas API (Interactive Starfield)
* **Icons:** FontAwesome 6.4.0
